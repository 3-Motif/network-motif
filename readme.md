# Network Motif
It counts the number of 3-node subgraphs in a given graph and displays the result visually. It also performs the count on the ensemble of randomly generated graphs compatible with the given network.

### Quick setup
1. Clone the repo
```sh
git clone git@github.com:Jeigsaw/motif-count.git
```
2. Open your terminal in the cloned directory and move into the app directory.

```sh
cd app
``` 
3. Start the local server.

- You need python installed on your machine or You can use any other method to start the local server
```sh
python -m http.server
```

### Disclaimer
For current use case, the compiled files are included in the repo. <br>
You can compile the cpp codes on your own using Emscripten.
